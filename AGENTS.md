# Idioma: español neutro (regla estricta)

Toda comunicación con el usuario y todo texto de UI deben estar en **español neutro**,
sin voseo ni regionalismos (no usar "vos / tenés / podés / dale / acá"; usar "tú puedes"
o formas impersonales, "aquí"). El usuario lo exige explícitamente.
